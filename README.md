# Automated-teller-machine-Simulation-app
ATM Simulation App built in pure Java using Swing and AWT components. Provides core ATM functionalities like user login, balance inquiry, deposit, withdrawal, and transaction history with a simple GUI interface. Designed for learning Java GUI and OOP concepts.


ATM Simulation App (Java Swing & AWT)
📌 Overview

This project is a pure Java-based ATM Simulation App developed using Swing and AWT components. It demonstrates the core functionality of an Automated Teller Machine (ATM) with a simple, user-friendly GUI.

✨ Features

🔐 User Login system

💰 Balance Inquiry

➕ Deposit Money

➖ Withdraw Money

📜 Transaction History

❌ Logout / Exit option

🛠️ Tech Stack

Language: Java (Core Java Concepts)

GUI: Swing & AWT

Paradigm: Object-Oriented Programming (OOP)

🎯 Learning Outcomes

Understanding of Java Swing & AWT for GUI design

Implementation of OOP principles in Java

Handling basic event-driven programming

Designing user-friendly desktop applications
